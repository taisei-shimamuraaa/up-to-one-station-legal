# 1駅だけ｜法務ページ

iOS / Android アプリ **「1駅だけ」** の法務ページ一式（プライバシーポリシー・利用規約・特定商取引法に基づく表記）です。
GitHub Pages で公開し、App Store Connect / Google Play Console に URL を登録して利用します。

## ファイル構成

```
.
├── index.html        トップページ
├── privacy.html      プライバシーポリシー
├── terms.html        利用規約
├── tokushoho.html    特定商取引法に基づく表記
├── style.css         共通スタイル
└── README.md
```

静的 HTML / CSS のみで構成されており、フレームワーク・JavaScript は使用していません。

## 内容を更新する

1. 対象の HTML を編集
2. コミットして `main` に push
3. GitHub Pages により自動で再デプロイ（数十秒〜数分）

利用規約・プライバシーポリシーを実質的に変更した場合は、各ページ末尾の **制定日 / 最終更新日** も更新してください。
