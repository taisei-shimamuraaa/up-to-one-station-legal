# 前提

アプリ名：1駅だけ

重要：
このアプリは、医療行為、診断、治療、医学的助言の提供を目的としたものではありません。
症状の改善、治療効果、特定の結果を保証するものではありません。
ユーザーには、体調や安全に配慮し、無理のない範囲で利用してもらう前提です。
体調に不安がある場合、症状が強い場合、緊急性がある場合は、医療機関、専門家、または緊急相談窓口への相談を促します。

- ただし色は強すぎないようにしてください
